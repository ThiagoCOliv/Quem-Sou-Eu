# Quem Sou Eu

Repositório pessoal para reunir currículos, certificados e materiais que documentam minha evolução profissional.

## Estrutura

- `Curriculos/` — versões do meu currículo em Markdown (ex.: "THIAGO - Curriculo Geral.md", "THIAGO CORREA - Curriculo Dev.md", "THIAGO - Curriculo Consolidado.md").
- `Certificados/` — certificados e imagens de cursos e bootcamps concluídos.
- `README.md` — este arquivo com visão geral e instruções rápidas.

## Conteúdo principal

- Currículos em formatos editáveis (Markdown) para manutenção e conversão
	para PDF quando necessário.
- Certificados organizados por plataforma e/ou evento.
